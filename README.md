## ts-doc-gen

cli for generating markdown docs from `d.ts` files.

### Usage

just run `tdg`.

| Flag                | Default     | Description                                      |
| ------------------- | ----------- | ------------------------------------------------ |
| `--out OUTPUT_FILE` | `README.md` | Relative path to the output file with extension. |
| `--dir DIR`         | `dist`      | Relative path to the input dir.                  |
